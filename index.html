Projeto mais recente.

No meu projeto de conclusão de curso, estou implementando um sistema Web com a finalidade de unir estudantes com necessidade de produção acadêmica (como seu trabalho de conclusão de curso, iniciação científica e projeto de extensão) com os docentes que os propõem.
 Neste projeto, utilizarei Django no back-end, Vue.js no front-end e um banco de dados MongoDB.
A decisão de escolher esses frameworks se deu pelo fato de que desejo que meu sistema seja posteriormente implantado e utilizado na minha instituição,
ajudando alunos com a mesma necessidade que eu tive desde que iniciei o meu curso.
 Este trabalho de conclusão está sendo orientado pelo professor Paulo Abadie Guedes.
 Utilizarei a biblioteca Jest para testar meus componentes front-end e o unittest para testar a lógica do meu servidor.
